Get started with Kotlin

Kotlin is a modern but already mature programming language aimed to make developers happier. It’s concise, safe, interoperable with Java and other languages, and provides many ways to reuse code between multiple platforms for productive programming. Pick it up to start building powerful applications!

package test.bar import foo.bar import footoo.bar as footoo

Kotlin for server side

Kotlin is a great fit for developing server-side applications. It allows you to write concise and expressive code while maintaining full compatibility with existing Java-based technology stacks, all with a smooth learning curve:

  • IExpressiveness: Kotlin's innovative language features, such as its support for type-safe builders and delegated properties, help build powerful and easy-to-use abstractions.
  • Scalability: Kotlin's support for coroutines helps build server-side applications that scale to massive numbers of clients with modest hardware requirements.
  • Interoperability: Kotlin is fully compatible with all Java-based frameworks, so you can use your familiar technology stack while reaping the benefits of a more modern language.
  • Migration: Kotlin supports gradual migration of large codebases from Java to Kotlin. You can start writing new code in Kotlin while keeping older parts of your system in Java.
  • Tooling: In addition to great IDE support in general, Kotlin offers framework-specific tooling (for example, for Spring) in the plugin for IntelliJ IDEA Ultimate.
  • Learning Curve: For a Java developer, getting started with Kotlin is very easy. The automated Java-to-Kotlin converter included in the Kotlin plugin helps with the first steps. Kotlin Koans can guide you through the key features of the language with a series of interactive exercises.
Kotlin para Android

El desarrollo móvil Android ha sido el primero de Kotlin desde Google I / O en 2019. Usando Kotlin para el desarrollo de Android, puede beneficiarse de:

  • Menos código combinado con mayor legibilidad. Pasa menos tiempo escribiendo tu código y trabajando para entender el código de los demás.
  • Lenguaje y entorno maduros. Desde su creación en 2011, Kotlin se ha desarrollado continuamente, no solo como un lenguaje sino como un ecosistema completo con herramientas robustas. Ahora está perfectamente integrado en Android Studio y es utilizado activamente por muchas empresas para desarrollar aplicaciones androides.
  • Soporte de Kotlin en Android Jetpack y otras bibliotecas. Las extensiones KTX agregan características del lenguaje Kotlin, como corutinas, funciones de extensión, lambdas y parámetros con nombre, a las bibliotecas de Android existentes.
  • Interoperabilidad con Java. Puede utilizar Kotlin junto con el lenguaje de programación Java en sus aplicaciones sin necesidad de migrar todo su código a Kotlin.
  • Soporte para desarrollo multiplataforma. Puede usar Kotlin para desarrollar no solo Android, sino también iOS,backend y aplicaciones web. Disfruta de los beneficios de compartir el código común entre las plataformas.
  • Código de seguridad. Menos código y mejor legibilidad conducen a menos errores. El compilador Kotlin detecta estos errores restantes, haciendo que el código sea seguro.
  • Fácil aprendizaje. Kotlin es muy fácil de aprender, especialmente para los desarrolladores de Java.

Kotlin para JavaScript

Kotlin/JS proporciona la capacidad de transpilar su código Kotlin, la biblioteca estándar Kotlin y cualquier dependencia compatible con JavaScript. La implementación actual de Kotlin/JS apunta a ES5. La forma recomendada de usar Kotlin / JS es a través de los complementos y Gradle. Le permiten configurar y controlar fácilmente los proyectos de Kotlin dirigidos a JavaScript en un solo lugar. Esto incluye funcionalidades esenciales como controlar el empaquetado de la aplicación, agregar dependencias de JavaScript directamente desde npm y más. Para obtener una visión general de las opciones disponibles, consulte la documentación de configuración del proyecto Kotlin/JS.kotlin.jskotlin.multiplatform

Kotlin para la ciencia de datos

Desde la creación de canalizaciones de datos hasta la producción de modelos de aprendizaje automático, Kotlin puede ser una excelente opción para trabajar con datos:

  • Kotlin es conciso, legible y fácil de aprender.
  • La escritura estática y la seguridad nula ayudan a crear código confiable y mantenible que es fácil de solucionar.
  • Al ser un lenguaje JVM, Kotlin le brinda un gran rendimiento y la capacidad de aprovechar todo un ecosistema de bibliotecas Java probadas y verdaderas.
Kotlin para programación competitiva

Este tutorial está diseñado tanto para programadores competitivos que no usaron Kotlin antes como para desarrolladores de Kotlin que no participaron en ningún evento de programación competitiva antes. Asume las habilidades de programación correspondientes. La programación competitiva es un deporte mental donde los concursantes escriben programas para resolver problemas algorítmicos especificados con precisión dentro de estrictas restricciones. Los problemas pueden variar desde los simples que pueden ser resueltos por cualquier desarrollador de software y requieren poco código para obtener una solución correcta, hasta los complejos que requieren conocimiento de algoritmos especiales, estructuras de datos y mucha práctica. Si bien no está diseñado específicamente para la programación competitiva, Kotlin encaja bien en este dominio, reduciendo la cantidad típica de repetitivos que un programador necesita escribir y leer mientras trabaja con el código casi al nivel ofrecido por los lenguajes de scripting de tipo dinámico, mientras que tiene herramientas y rendimiento de un lenguaje de tipo estático. Consulte Introducción a Kotlin/JVM sobre cómo configurar el entorno de desarrollo para Kotlin. En la programación competitiva, generalmente se crea un solo proyecto y la solución de cada problema se escribe en un solo archivo fuente. tailrec fun removeZeroes(x: Int): Int = if (x % 10 == 0) removeZeroes(x / 10) else x fun f(x: Int) = removeZeroes(x + 1)

Kotlin Multiplataforma

El soporte para la programación multiplataforma es uno de los beneficios clave de Kotlin. Reduce el tiempo dedicado a escribir y mantener el mismo código para diferentes plataformas, al tiempo que conserva la flexibilidad y los beneficios de la programación nativa. Obtenga más información sobre los beneficios multiplataforma de Kotlin. Con Kotlin Multiplatform, comparta el código utilizando los mecanismos que Kotlin proporciona:

  • La creación de una biblioteca Kotlin multiplataforma enseña cómo crear una biblioteca multiplataforma disponible para JVM, JS y Native y que se puede usar desde cualquier otro código común (por ejemplo, compartido con Android e iOS). También muestra cómo escribir pruebas que se ejecutarán en todas las plataformas y utilizar una implementación eficiente proporcionada por una plataforma específica.
  • La creación de una aplicación web full stack con Kotlin Multiplatform enseña los conceptos detrás de la creación de una aplicación que se dirige a Kotlin / JVM y Kotlin / JS mediante la creación de una aplicación cliente-servidor que hace uso de código compartido, serialización y otros paradigmas multiplataforma. También proporciona una breve introducción al trabajo con Ktor como un marco del lado del servidor y del cliente.
Ejecutar pruebas en Kotlin/JS

El complemento Kotlin / JS Gradle le permite ejecutar pruebas a través de una variedad de corredores de prueba que se pueden especificar a través de la configuración de Gradle. Para que las anotaciones de prueba y la funcionalidad estén disponibles para el destino javascript, agregue el artefacto de plataforma correcto para kotlin.test en :build.gradle.kts target { browser { testTask { useKarma { useChromeHeadless() useFirefox() } } } }
